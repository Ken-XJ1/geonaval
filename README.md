# GEONAVAL — Sistema de Registro Fluvial y Monitoreo GPS

Sistema web para el control y monitoreo del transporte fluvial en Quibdó, Chocó, Colombia (Río Atrato).

## FIGMA — Diseño de la interfaz

[Ver diseño en Figma](https://www.figma.com/design/u0Z8YPiRcr7QpizacVXhJN/Interfaz-web-para-GEONAVAL--copia-)

## Sitio web en producción

[https://geonaval.onrender.com](https://geonaval.onrender.com)

## Stack tecnológico

- **Frontend:** React + Vite + TypeScript + shadcn/ui + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Base de datos:** PostgreSQL
- **Hosting:** Render.com
- **Autenticacion:** JWT + bcryptjs

## Roles del sistema

| Rol | Descripcion |
|-----|-------------|
| Administrador | Acceso completo al sistema |
| Operador | Gestion de sus viajes asignados |
| Cliente | Inscripcion y seguimiento de viajes |
| Autoridad | Supervision y reportes oficiales |

## Instalacion local

npm install
npm run dev

## Modulos

- Gestion de embarcaciones
- Gestion de propietarios
- Gestion de tripulacion
- Gestion de pasajeros
- Gestion de viajes (zarpe)
- Monitoreo GPS en tiempo real
- Reportes y estadisticas
- Acceso para autoridades
