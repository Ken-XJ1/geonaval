Diseña una plataforma web moderna, profesional, segura y responsive llamada “GEONAVAL”, enfocada en el control y monitoreo del transporte fluvial en Quibdó, Chocó, Colombia.

El sistema debe tener control de acceso por roles, donde cada usuario vea únicamente la información que le corresponde según sus permisos.

⚠️ IMPORTANTE:
NO todos los usuarios deben tener acceso a toda la información.
El Administrador tendrá acceso completo.
Los demás usuarios tendrán acceso limitado según su función.
